# Donate

**คำอธิบาย:** ระบบจัดการคำขอมอบเงินแบบ Session-Based

## สรุปสั้น ๆ
ระบบ Donate จัดการ lifecycle ของคำขอการมอบเงินในรูปแบบ session-based โดยรองรับการสร้างคำขอ การตรวจสอบ/อนุมัติแบบหลายขั้นตอน การเชื่อมต่อกับระบบชำระเงินเพื่อดำเนินการมอบเงิน และการบันทึกประวัติสำหรับการตรวจสอบ

## คุณสมบัติเด่น
- Session-based lifecycle สำหรับคำขอการมอบเงิน
- Multi-step approval และ role-based access control
- Integration point สำหรับการมอบเงินผ่านระบบชำระเงินภายนอก
- Audit trail และ reporting

## การใช้งานด่วน (Conceptual Quickstart)
1. สร้าง session ผ่าน API หรือ UI ของระบบ
2. ระบบแจ้งให้ผู้ตรวจสอบทำการพิจารณา
3. เมื่ออนุมัติ ระบบเชื่อมต่อกับระบบชำระเงินเพื่อมอบเงิน
4. ตรวจสอบสถานะและประวัติผ่าน admin dashboard

> หมายเหตุ: เอกสารนี้เป็นคู่มือเชิงแนวคิด — สำหรับขั้นตอนการติดตั้งจริงและการตั้งค่าพารามิเตอร์ โปรดดู USER_GUIDE.md

## การผสานงานกับ PSAistudio
ระบบ Donate ออกแบบให้ผสานงานกับ PSAistudio เพื่อเรียกสร้าง payment request และรับผลการชำระผ่าน webhook เพื่อนำมาปิด session
